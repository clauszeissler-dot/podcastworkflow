## 1. E-Mail-Entwurf mit Kontext

```
Schreibe eine professionelle E-Mail an [Empfänger/Rolle].

Kontext: [Beschreibe die Situation]
Ziel: [Was soll erreicht werden?]
Tonalität: [formell/freundlich/direkt]
Maximale Länge: [kurz/mittel/ausführlich]

Füge eine passende Betreffzeile hinzu.
```

**Beispiel:**
> Schreibe eine professionelle E-Mail an meinen Projektleiter.
> Kontext: Ich brauche eine Fristverlängerung für den Quartalsbericht
> Ziel: 3 zusätzliche Tage bekommen
> Tonalität: freundlich aber bestimmt
> Maximale Länge: kurz

---

## 2. Meeting-Protokoll aus Notizen

```
Erstelle ein strukturiertes Meeting-Protokoll aus folgenden Stichpunkten:

[Füge deine Notizen ein]

Format:
- Datum & Teilnehmer
- Besprochene Themen (mit Kernaussagen)
- Entscheidungen
- Action Items (Wer macht was bis wann?)
- Nächste Schritte
```

---

## 3. Komplexe Themen einfach erklären

```
Erkläre [komplexes Thema] so, dass es [Zielgruppe] in 2 Minuten verstehen kann.

Verwende:
- Maximal 3 Kernpunkte
- Eine Analogie aus dem Alltag
- Keine Fachbegriffe (oder erkläre sie sofort)

Zielgruppe: [z.B. Kollegen ohne technischen Hintergrund]
```

---

## 4. Feedback konstruktiv formulieren

```
Formuliere konstruktives Feedback zu folgender Situation:

Was ist passiert: [Beschreibung]
Was war problematisch: [konkret benennen]
Gewünschtes Ergebnis: [was soll sich ändern]

Das Feedback soll:
- Wertschätzend beginnen
- Konkret und sachlich sein
- Einen Lösungsvorschlag enthalten
- Motivation erhalten
```

---

## 5. Präsentations-Gliederung erstellen

```
Erstelle eine Gliederung für eine [Dauer]-minütige Präsentation zum Thema:

[Thema]

Zielgruppe: [wer hört zu?]
Ziel der Präsentation: [informieren/überzeugen/aktivieren]
Kernbotschaft: [was soll hängen bleiben?]

Struktur mit Zeitangaben pro Folie/Abschnitt.
Schlage 3 mögliche Einstiegshooks vor.
```

---

## 6. Aufgaben priorisieren (Eisenhower-Matrix)

```
Sortiere folgende Aufgaben nach der Eisenhower-Matrix:

[Liste deine Aufgaben auf]

Kategorisiere in:
1. DRINGEND + WICHTIG → Sofort erledigen
2. WICHTIG, nicht dringend → Terminieren
3. DRINGEND, nicht wichtig → Delegieren
4. Weder noch → Streichen

Begründe kurz jede Einordnung.
```

---

## 7. Schwierige Gespräche vorbereiten

```
Hilf mir, ein schwieriges Gespräch vorzubereiten:

Situation: [was ist der Anlass?]
Gesprächspartner: [Rolle/Beziehung]
Mein Ziel: [was will ich erreichen?]
Meine Sorge: [was könnte schiefgehen?]

Erstelle:
1. Gesprächseröffnung (2-3 Sätze)
2. Kernargumente (max. 3)
3. Mögliche Einwände + Antworten
4. Plan B falls es eskaliert
```

---

## 8. Texte kürzen ohne Inhaltsverlust

```
Kürze folgenden Text auf [Ziel-Wortanzahl] Wörter, ohne wichtige Informationen zu verlieren:

[Text einfügen]

Behalte:
- Alle Fakten und Zahlen
- Die Kernaussage
- Den professionellen Ton

Entferne:
- Füllwörter
- Redundanzen
- Unnötige Adjektive
```

---

## 9. Brainstorming-Partner

```
Ich brauche Ideen für: [Problem/Herausforderung]

Kontext: [relevante Hintergrundinformationen]
Einschränkungen: [Budget/Zeit/Ressourcen]
Bisherige Versuche: [was wurde schon probiert?]

Generiere 10 unterschiedliche Lösungsansätze:
- 3 konservative/sichere
- 4 kreative/unkonventionelle
- 3 "verrückte" (die trotzdem funktionieren könnten)

Bewerte jeden Ansatz kurz nach Aufwand und Erfolgschance.
```

---

## 10. Wochenbericht erstellen

```
Erstelle einen Wochenbericht aus folgenden Stichpunkten:

Diese Woche erledigt:
[Liste]

In Arbeit:
[Liste]

Blockiert/Probleme:
[Liste]

Format:
- Executive Summary (3 Sätze)
- Highlights der Woche
- Kennzahlen (falls relevant)
- Ausblick nächste Woche
- Unterstützungsbedarf

Tonalität: professionell, ergebnisorientiert
```

---

## Bonus-Tipp: Der Meta-Prompt

```
Ich möchte [Ziel erreichen].

Stelle mir die 5 wichtigsten Fragen, die du brauchst,
um mir dabei optimal zu helfen.
```

*Dieser Prompt funktioniert immer, wenn du nicht weißt, wie du anfangen sollst!*
