## Wichtiger Hinweis für Eltern

Diese Prompts sind dafür gedacht, **gemeinsam** mit Kindern genutzt zu werden.
KI ist ein Werkzeug - wie ein Taschenrechner oder ein Lexikon.
Begleiten Sie Ihr Kind und besprechen Sie die Ergebnisse zusammen.

---

## 1. Hausaufgaben-Erklärer (nicht Löser!)

```
Mein Kind ([Alter] Jahre, [Klasse]) versteht folgendes Thema nicht:

[Thema/Aufgabe beschreiben]

Erkläre es so, dass ein Kind es verstehen kann:
- Verwende eine Alltagsanalogie
- Gehe Schritt für Schritt vor
- Gib ein einfaches Beispiel
- Dann ein etwas schwierigeres

WICHTIG: Gib NICHT die Lösung, sondern erkläre den Weg!
Am Ende: 2 Übungsaufgaben zum Selbermachen.
```

**Hinweis:** Die KI soll erklären, nicht die Hausaufgaben machen!

## 2. Gute-Nacht-Geschichte Generator

```
Erfinde eine Gute-Nacht-Geschichte für mein Kind:

Name des Kindes: [Name]
Alter: [Jahre]
Lieblingstier/-figur: [z.B. Drache, Einhorn, Astronaut]
Aktuelles Thema/Interesse: [z.B. Dinosaurier, Prinzessinnen, Fußball]
Soll die Geschichte eine Botschaft haben? [z.B. Mut, Freundschaft, Teilen]

Länge: ca. [5/10/15] Minuten Vorlesezeit
Gruselfaktor: [keiner/ein bisschen spannend]

Die Geschichte soll:
- [Name] als Hauptfigur haben
- Ein kleines Abenteuer sein
- Gut ausgehen
- Mit einer kuscheligen Szene enden
```

## 3. Kinderfragen beantworten

```
Mein Kind ([Alter]) hat gefragt: "[Die Frage des Kindes]"

Beantworte die Frage:
- Altersgerecht und ehrlich
- Mit einem anschaulichen Vergleich
- In maximal 5 einfachen Sätzen
- Ohne zu vereinfachen, aber verständlich

Am Ende: Eine Gegenfrage, die zum Weiterdenken anregt.

Falls die Frage sensibel ist, gib mir auch einen Tipp,
wie ich das Gespräch weiterführen kann.
```

## 4. Bastel- & Spielideen bei Langeweile

```
Wir brauchen eine Beschäftigung!

Anzahl Kinder: [Anzahl]
Alter: [von-bis Jahre]
Drinnen oder draußen: [drinnen/draußen/egal]
Verfügbare Zeit: [Minuten]
Material vorhanden: [Standard-Haushalt/Bastelsachen/wenig]
Wetter: [sonnig/regnerisch/etc.]

Generiere 5 Ideen:
1. Eine Bastel-Aktivität
2. Ein Bewegungsspiel
3. Ein Lernspiel (das sich nicht wie Lernen anfühlt)
4. Etwas Kreatives
5. Ein Experiment/Entdeckung

Mit Anleitung und Materialliste!
```

## 5. Lernspiele für unterwegs

```
Wir haben eine [Auto/Zug/Flug]-Reise von [Dauer].

Kinder dabei: [Alter der Kinder]
Verfügbare Hilfsmittel: [nur Reden/Papier+Stift/Handy erlaubt]

Gib mir 10 Spiele/Aktivitäten:
- Die keine Vorbereitung brauchen
- Die nicht nervig für Erwachsene sind
- Die verschiedene Schwierigkeiten haben
- Mit klaren, kurzen Regeln

Sortiere nach: Ruhig → Aktiver
```

## 6. Referat & Präsentation vorbereiten

```
Mein Kind (Klasse [X]) muss ein Referat halten über:

Thema: [Thema]
Länge: [Minuten]
Hilfsmittel erlaubt: [Plakat/PowerPoint/nur Sprechen]

Hilf uns bei:
1. Gliederung (kindgerecht, mit Zeitangaben)
2. 5 interessante Fakten, die beeindrucken
3. Eine spannende Einstiegsfrage
4. Einfache Erklärungen für schwierige Begriffe
5. Ideen für Bilder/Zeichnungen
6. Ein Quiz am Ende für die Klasse (3 Fragen)

Sprache: So dass [Alter]-Jährige es verstehen UND erklären können.
```

## 7. Streit schlichten helfen

```
Meine Kinder streiten sich gerade über:

[Situation beschreiben]

Beteiligte: [Alter der Kinder]
Das sagt Kind 1: [Position]
Das sagt Kind 2: [Position]

Hilf mir:
1. Beide Seiten zu verstehen
2. Fairness-Fragen, die ich stellen kann
3. 3 mögliche Kompromisse
4. Wie ich das Gespräch leiten kann
5. Eine "Friedensregel" für die Zukunft

Ich will nicht richten, sondern beide zum Nachdenken bringen.
```

## 8. Kindgerechte Erklärung von Weltereignissen

```
Mein Kind ([Alter]) hat von [Thema/Ereignis] gehört und fragt danach.

Kontext: [Woher weiß es davon? Was hat es gehört?]
Meine Sorge: [Was will ich vermeiden?]

Erstelle eine kindgerechte Erklärung:
- Ehrlich, aber nicht ängstigend
- Mit Fokus auf das, was Menschen Gutes tun
- Ohne zu viele beunruhigende Details
- Mit Raum für Gefühle

Füge hinzu:
- Was wir als Familie tun können (falls passend)
- Gesprächsangebote für weitere Fragen
```

## 9. Vokabeln & Fakten spielerisch lernen

```
Mein Kind muss lernen:

Fach: [Englisch-Vokabeln/Sachkunde-Fakten/Mathe-Einmaleins/etc.]
Konkret: [Was genau? Z.B. Vokabelliste, Bundesländer, etc.]
Alter: [Jahre]

Erstelle:
1. Ein Memory-Spiel (Paare zum Ausdrucken)
2. 5 Eselsbrücken/Merksprüche
3. Ein Quiz mit 10 Fragen (Multiple Choice)
4. Ein Bewegungsspiel zum Lernen
5. Eine Geschichte, in der die Fakten vorkommen

Mach es so, dass es sich nicht wie Lernen anfühlt!
```

## 10. Geburtstags-Party Planer

```
Wir planen einen Kindergeburtstag:

Geburtstagskind: [Alter]
Motto/Thema: [falls vorhanden]
Anzahl Gäste: [Anzahl Kinder]
Alter der Gäste: [von-bis]
Location: [Zuhause/Garten/Ort]
Dauer: [Stunden]
Budget: ca. [Betrag] €

Plane:
1. Zeitablauf (Ankommen bis Abholen)
2. 3 Spiele (mit Anleitung)
3. Schatzsuche/Schnitzeljagd Konzept
4. Einfache Deko-Ideen (DIY)
5. Essen & Kuchen (kindertauglich)
6. Mitgebsel-Ideen
7. Plan B bei schlechtem Wetter

Checkliste: Was muss wann vorbereitet werden?
```

## Bonus: Der "Warum?"-Beantworter

Wenn dein Kind in der "Warum?"-Phase ist:

```
Mein Kind fragt immer weiter "Warum?".

Die Kette bisher:
- Warum ist der Himmel blau?
- [Deine Antwort] → Warum?
- [Deine Antwort] → Warum?

Gib mir die nächsten 3 "Warum"-Antworten,
die kindgerecht sind und irgendwann bei einer
zufriedenstellenden Antwort enden.

Beende mit etwas Staunen-Wertem!
```

### Für Eltern: Tipps zur KI-Nutzung mit Kindern

1. **Gemeinsam nutzen** - Lassen Sie Ihr Kind nicht allein mit der KI
2. **Kritisch bleiben** - Besprechen Sie: "Stimmt das? Wie können wir das prüfen?"
3. **Kreativität fördern** - KI ist Ausgangspunkt, nicht Endprodukt
4. **Grenzen setzen** - Bildschirmzeit gilt auch für KI-Nutzung
5. **Vorbild sein** - Zeigen Sie verantwortungsvollen Umgang
