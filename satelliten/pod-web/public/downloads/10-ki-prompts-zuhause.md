## 1. Wochenplan für Mahlzeiten

```
Erstelle einen Essensplan für [Anzahl] Personen für die kommende Woche.

Rahmenbedingungen:
- Budget pro Woche: ca. [Betrag] €
- Kochzeit pro Mahlzeit: maximal [Zeit] Minuten
- Ernährungsform: [normal/vegetarisch/vegan/etc.]
- Allergien/Unverträglichkeiten: [falls vorhanden]

Bitte erstelle:
1. Tagesplan (Mo-So, Mittag + Abend)
2. Einkaufsliste (nach Supermarkt-Abteilungen sortiert)
3. Meal-Prep-Tipps (was kann ich vorkochen?)

Bonus: Nutze Zutaten mehrfach, um Reste zu vermeiden.
```

## 2. Reiseplanung mit Insider-Tipps

```
Plane eine [Dauer]-tägige Reise nach [Zielort].

Reisende: [Anzahl Erwachsene/Kinder]
Budget: [Rahmen]
Interessen: [z.B. Kultur, Natur, Essen, Abenteuer]
Reisezeit: [Monat]

Erstelle:
1. Tagesplan mit Aktivitäten
2. Unterkunftsempfehlungen (3 Optionen: Budget/Mittel/Komfort)
3. Restauranttipps (inkl. lokale Spezialitäten)
4. Geheimtipps abseits der Touristenpfade
5. Packliste für das Wetter
6. Ungefähre Kostenaufstellung
```

## 3. Geschenkideen-Generator

```
Ich suche ein Geschenk für [Beziehung: Mutter/Freund/Kollegin etc.].

Über die Person:
- Alter: [ca.]
- Interessen/Hobbys: [was bekannt ist]
- Was sie definitiv NICHT mag: [falls bekannt]
- Letztes Geschenk von mir: [falls relevant]

Budget: [Betrag] €
Anlass: [Geburtstag/Weihnachten/Danke/etc.]

Generiere 10 Ideen:
- 3 klassische/sichere Optionen
- 3 Erlebnis-Geschenke
- 2 selbstgemachte/persönliche Ideen
- 2 außergewöhnliche Überraschungen

Mit Begründung, warum es passen könnte.
```

---

## 4. Reklamation/Beschwerde formulieren

```
Hilf mir, eine [freundliche aber bestimmte] Beschwerde zu schreiben.

Situation: [Was ist passiert?]
Unternehmen/Anbieter: [Name]
Was ich will: [Erstattung/Umtausch/Entschuldigung/etc.]
Belege vorhanden: [Rechnung/Fotos/etc.]

Die Nachricht soll:
- Sachlich bleiben (keine Emotionen)
- Fakten klar darstellen
- Meine Rechte nennen (falls relevant)
- Konkrete Forderung enthalten
- Frist setzen

Format: [E-Mail/Brief/Chat]
```

## 5. Fitness- & Bewegungsplan

```
Erstelle einen Bewegungsplan für mich:

Aktueller Fitness-Level: [Anfänger/Fortgeschritten/etc.]
Ziel: [Abnehmen/Muskelaufbau/Beweglicher werden/Stress abbauen]
Verfügbare Zeit pro Woche: [Stunden]
Equipment: [Nichts/Hanteln/Fitnessstudio/etc.]
Einschränkungen: [Knie/Rücken/etc. falls vorhanden]

Plane:
1. Wochenübersicht (welche Tage, wie lange)
2. Konkrete Übungen mit Wiederholungen
3. Aufwärm- und Cool-Down-Routine
4. Progression über 4 Wochen
5. Motivationstipps für schwache Momente
```

## 6. Verträge & Dokumente verstehen

```
Erkläre mir diesen [Vertrag/AGB/Brief] in einfacher Sprache:

[Text einfügen]

Ich möchte wissen:
1. Was sind meine Pflichten?
2. Was sind meine Rechte?
3. Gibt es versteckte Kosten oder Fallen?
4. Kündigungsfristen und -bedingungen?
5. Worauf sollte ich besonders achten?

Markiere kritische Stellen mit ⚠️
```

## 7. Haushalts-Problemlöser

```
Ich habe folgendes Problem im Haushalt:

[Beschreibe das Problem: Fleck, kaputtes Gerät, Schädlinge, etc.]

Bereits versucht: [was hat nicht funktioniert?]

Bitte gib mir:
1. Die wahrscheinlichste Ursache
2. DIY-Lösung Schritt für Schritt
3. Hausmittel-Alternative (falls möglich)
4. Wann sollte ich einen Profi rufen?
5. Wie vermeide ich das Problem in Zukunft?

Sicherheitshinweise nicht vergessen!
```

## 8. Lernen & Weiterbildung planen

```
Ich möchte [Fähigkeit/Thema] lernen.

Mein Ausgangslevel: [keine Ahnung/Grundkenntnisse/etc.]
Verfügbare Zeit: [Stunden pro Woche]
Lerntyp: [Videos/Bücher/Praxis/Kurse]
Budget: [kostenlos/bis X €]
Ziel: [Was will ich konkret können?]
Zeitrahmen: [In wie vielen Wochen/Monaten?]

Erstelle:
1. Lernpfad mit Meilensteinen
2. Empfohlene Ressourcen (kostenlos + bezahlt)
3. Praktische Übungen pro Phase
4. Wie ich meinen Fortschritt messen kann
```

---

## 9. Schwierige Gespräche im Privaten

```
Ich muss ein schwieriges Gespräch führen mit [Beziehung].

Thema: [Worum geht es?]
Mein Ziel: [Was soll sich ändern?]
Meine Sorge: [Was hält mich zurück?]
Vorgeschichte: [Relevanter Kontext]

Hilf mir mit:
1. Dem richtigen Zeitpunkt/Setting
2. Gesprächseröffnung (Ich-Botschaften)
3. Wie ich beim Thema bleibe
4. Umgang mit emotionalen Reaktionen
5. Kompromissvorschläge
6. Wie wir das Gespräch positiv beenden können
```

---

## 10. Entscheidungen treffen

```
Ich stehe vor folgender Entscheidung:

Option A: [beschreiben]
Option B: [beschreiben]
(Optional: Option C)

Wichtig für mich:
- Finanziell: [Priorität hoch/mittel/niedrig]
- Zeitlich: [Priorität]
- Emotional: [Priorität]
- Langfristig: [Priorität]

Erstelle:
1. Pro/Contra-Liste für jede Option
2. "Was wäre wenn"-Szenarien (Best/Worst Case)
3. Welche Fragen sollte ich mir noch stellen?
4. Deine Empfehlung basierend auf meinen Prioritäten
5. "Bauchgefühl-Check" - Fragen zur Selbstreflexion
```

```
Ich habe heute [Zeitrahmen] Zeit und muss folgendes erledigen:

[Liste deiner Aufgaben]

Priorisiere nach Dringlichkeit und gruppiere nach Ort/Art.
```
